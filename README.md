# Smart Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project focuses on customer segmentation using the K-Means Clustering algorithm. The goal is to group customers based on their purchasing behavior and demographic characteristics, enabling businesses to better understand their customers and create targeted marketing strategies.

The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, feature scaling, optimal cluster selection using the Elbow Method, customer segmentation, and visualization of the generated clusters.

---

## 🚀 Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Feature Scaling
- K-Means Clustering
- Elbow Method for Optimal Number of Clusters
- Customer Cluster Visualization
- Business Insights from Customer Segments

---

## 📂 Dataset
The project uses the **Smart Cart Customers** dataset, which contains customer demographic information, purchasing behavior, and spending-related features.
